# Terrain Rendering using SDL3 GPU
---
Based on rastertek's Direct3d11 Terrain series. Roughly.

## Dependencies
- SDL3
- GLM

## Toolset
- CMake 3.31+
- CPM.CMake
- Ninja build
- MSVC
- Windows SDK
